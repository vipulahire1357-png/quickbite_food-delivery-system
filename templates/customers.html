Customers are linked to orders via customer_id (Foreign Key).


Each customer can place multiple orders. The order count below reflects the live JOIN query on the database.

SQL Used:

SELECT c.*, COUNT(o.id) AS order_count
FROM Customers c
LEFT JOIN Orders o ON c.id = o.customer_id
GROUP BY c.id
